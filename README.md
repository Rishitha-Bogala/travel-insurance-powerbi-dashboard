# Travel Insurance Risk & Customer Insights Dashboard

## Project Overview

This Power BI dashboard provides a comprehensive analysis of travel insurance operations, customer demographics, risk levels, and agent performance.

The dashboard was developed to help stakeholders monitor policy performance, evaluate customer risk profiles, analyze agent productivity, and identify key business insights through interactive visualizations.
